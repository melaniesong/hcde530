# MP1 — Mini Project 1 (standalone folder)

## Run locally

1. Download **`380K_US_Restaurants.csv`** from [Kaggle](https://www.kaggle.com/datasets/kwxdata/380k-restaurants-mostly-usa-based/data) into this folder, **or** copy it from another week if you already have it.
2. Open **`week6_mp1_starter.ipynb`** and run all cells. The notebook creates:
   - **`restaurants_deduped.csv`** (deduplicated listings)
   - **`restaurants_cleaned.csv`** (deduplicated + rows with valid ratings)
3. Chart PNGs are generated by `fig.write_image(...)` in the notebook.

## GitHub note

The three CSV files above are **gitignored** because each exceeds GitHub’s **100 MB** file limit. Everything else in this folder (notebook, `mp1.md`, PNGs, config) is in the repo.
